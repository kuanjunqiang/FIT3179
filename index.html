The Choropleth Map

A Choropleth map is a type of thematic map which can show the quantitative data which related to country perfectly. The choropleth map above describes the total comfirmed cases of COVID-19 of each country and the range of comfirmed cases is 10 to 10000, user can move pointer to each country for checking the more detail cofirmed cases of the country.

The Stacked Bar Chart

The Stacked Bar Chart below shows the overall comfirmed cases around the world. From the chart below, we can that United States had an extremely high value of comfirmed cases compared to other country.

The Line Chart

The Line Chart below also shows the overall COVID-19 comfirmed cases rate from February to October. User can sort with different counntries and alos select the minimum positive cases.

The Rule Chart

It shows the Daily Positive of Covid-19 cases of each country and each day from January to October. The main purpose for this visualisation is to see when is the most peak moment of Covid-19 around the world.

The Area Graph

This visualisation had show the death cases/ comfirmed cases of each country. From this visualisation, we can know which country had done a great job in healing people who had get infected by Covid-19.
